# URL Shortener

A simple URL Shortener web application built using Django. This project allows users to convert long URLs into short, shareable links and redirect users to the original URL.

## Features

- Generate short URLs from long URLs
- Redirect users using short links
- Store URLs in a database
- Simple and user-friendly interface
- Built using Django

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
